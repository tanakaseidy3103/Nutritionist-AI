# 🍽️ Nutritionist-AI

## 日本語

Nutritionist-AI は、食材の写真から AI が内容を分析し、レシピを提案するブラウザアプリです。カメラ撮影または画像アップロードに対応し、履歴保存や検索、好みの設定まで含めた実用的なデモとして設計しました。

このプロジェクトはハッカソン向けに作成したもので、フロントエンド、生成AI、認証、データ保存、UX改善を組み合わせたポートフォリオ向けの題材として使いやすいです。


### 主な機能

- デバイスのカメラで食材を撮影
- 画像アップロードにも対応
- AIによる食材分析
- 3つのおすすめレシピを表示
- レシピを履歴に保存
- 検索・フィルター機能
- 保存済みレシピ数の表示
- 分析中のローディング表示
- 履歴クリアボタン
- ミニ画像とトースト通知によるUX向上
- 好み・アレルギー設定
- 日本語、ポルトガル語、英語の切り替え

### 技術スタック

- HTML, CSS, JavaScript
- OpenAI API
- Firebase Auth, Firestore, Analytics
- Service Worker を使った PWA 対応

### 履歴書向けの一文

食材画像を AI で解析し、レシピ提案・履歴管理・ユーザー設定まで備えた Web アプリ「Nutritionist-AI」を開発しました。

### ローカル実行

## English

Nutritionist-AI is a browser-based food analysis app that uses AI to inspect ingredients from a photo and suggest recipes. It supports camera capture and image upload, and includes recipe history, search, filters, and user preference settings.

The project was created for a hackathon and works well as a portfolio piece because it combines front-end development, generative AI, authentication, data persistence, and UX improvements in one application.

### Repository

- GitHub: https://github.com/tanakaseidy3103/Nutritionist-AI.git

### Key Features

- Capture ingredients with the device camera
- Support image upload from gallery
- Analyze ingredients with AI
- Show 3 recipe recommendations
- Save recipes to history
- Search and filter saved recipes
- Display saved recipe count
- Show loading feedback during analysis
- Clear history with one click
- Improve UX with thumbnails and toast notifications
- Store preferences and allergies
- Switch between Japanese, Portuguese, and English UI

### Tech Stack

- HTML, CSS, JavaScript
- OpenAI API
- Firebase Auth, Firestore, Analytics
- PWA support with Service Worker
